Logo rules
[ and ] are always #6366f1 (indigo-500) — on any background.
sage matches primary text: #f1f5f9 on dark, #0d1526 on light. Weight 500.
SOUNDS is primary text color. Weight 800, tracking -0.01em, size 1.41× the [sage] size.
Never use all-caps, never alter the bracket characters, never change SOUNDS to title case.

Motion
Pad press
Scale to 0.93 on press / fire, transition transform 0.07s. Hold for 700ms then release.
Waveform bars
3 bars animated with waveform-bar keyframe (3px → 10px height), staggered at 0s / 0.06s / 0.13s, 0.36s ease-in-out alternate loop. Visible only during pad fire.
Glow on fire
Dark mode: box-shadow: 0 0 18px rgba(color, 0.30–0.35) using the pad's accent color. Light mode: 0 0 14px rgba(color, 0.20). Transitions with box-shadow 0.1s.
UI interactions
Background: 0.15s. Border-color: 0.1s. Transform: 0.07s. All ease default. No transitions on color/opacity beyond these.
